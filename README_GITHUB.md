# 明星撞脸测试网站

一个有趣的互动网站，用户上传照片后可以测试自己最像哪位明星。

## 🌟 功能特点

- 📸 照片上传（支持拖拽）
- ⭐ 性别筛选（男生/女生/不限）
- 🎯 智能匹配算法
- 📱 响应式设计
- ✨ 精美UI界面

## 🚀 部署到GitHub Pages

1. **创建GitHub仓库**
   - 在GitHub上新建一个仓库
   - 上传所有文件到仓库

2. **启用GitHub Pages**
   - 进入仓库Settings
   - 找到Pages选项
   - 选择Source为main分支
   - 保存设置

3. **访问网站**
   - 几分钟后即可通过 `https://用户名.github.io/仓库名` 访问
